# 概要
下記の機能を追加してください。
追加時は、今まであった機能で、追加機能とは関係ないものは消さないようにしてください。
機能確認済みの項目には[×]と記載するので、それについては無視してほしい。

## グラフ
- [ ] 縦のスクロールで、一番上の項目は消えないように、固定してほしい。
- [ ] 横のスクロールも、左の項目(ポイントまで)は、固定してほしい。
- [ ] 左の項目(大項目、中項目、小項目、ポイント)の幅を変更できるようにしてほしい。
- [ ] 依存関係リンクモードで、始点をクリックする状態、終点をクリックする状態、などの作業者が次にやるべき作業がわかるようにしてほしい。
- [ ] 依存関係リンクモードで、戻ることはできるが、進むを押しても、状態が遷移しないため、改善してほしい。
- [ ] ややこしいので、大項目、中項目の背景色は、縞々にせずに、白で統一してほしい。

## 動作設定
- [ ] 一番左のNoの列を並び替えると、112334579のように、なぜか同じNoになってしまいます。これは不具合なので、重複しないように改善してほしい。Noも横の行と紐づけていればこんなことにはならないはず。
- [ ] 項目の並び順ですが、No、動作UID、大項目、中項目、小項目、開始ポイント、開始トリガ、開始依存元UID、終了ポイント、終了設定、終了トリガ、終了依存元UID、時間(ms)の順にしてほしい。単純に並び替え。
- [ ] また、見やすくするために、No、動作UID、大項目、中項目、小項目の上は「項目」、開始ポイント、開始トリガ、開始依存元UIDの上は「開始設定」、終了ポイント、終了設定、終了トリガ、終了依存元UID、時間(ms)の上は「終了設定」として、表示を追加してほしい。
- [ ] 開始ポイントには、未設定(-)を追加して、未設定の場合は、時系列的に現在の状態から開始できるようにしてほしい。
- [ ] onoff制御する場合、開始がOFFで、終了がONなら、開始時にOFFにして、設定時間や終了条件を満たしたときに、ONにしてほしい。
